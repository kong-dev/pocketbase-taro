## Installation

```sh
# npm
npm install pocketbase-taro
# yarn
yarn add pocketbase-taro
# pnpm
pnpm add pocketbase-taro
```

## Usage

```diff
- import PocketBase from 'pocketbase'
+ import PocketBase from 'pocketbase-taro'

const pb = new PocketBase('http://127.0.0.1:8090')
```
